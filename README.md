# Singku-Cafe-

Project Type: Mockup Website for Café Ordering and Management

Description:
Singku Café is an interactive web mockup designed to showcase an online café experience combining customer engagement, menu exploration, and management tools. The website allows visitors to explore the café’s offerings, view features, and simulate placing orders, while also providing an admin interface to manage customer records and messages. The design emphasizes a warm, inviting aesthetic with a minimalist layout, consistent branding, and interactive elements to reflect the café’s cozy and creative atmosphere.

Key Pages and Their Functions:
- Home Page: Introduces the café, highlights signature drinks, showcases reviews, and promotes key attractions like the Singku Rooftop.
- Features Page: Showcases Singku Café’s unique aspects, including community events, live music nights, sustainability initiatives, and brand values.
- Menu Page: Allows users to browse items, add them to a cart, and simulate checkout, reflecting an interactive ordering experience.
- Contact Page: Provides a communication hub with feedback forms, inquiries, and a map to the café location.
- Customer Records Page: Displays a table of customers with search and filter options for admin tracking and management.
- Messages Management Page: Simulates viewing messages from customers, including search and filtering capabilities for staff/admin use.

Website Features:
- Interactive menu with “Add to Cart” and checkout simulation.
- Customer and message management mockup pages for staff/admin.
- Search and filter options for efficient navigation.
- Visual emphasis on café branding, community events, and sustainability.
- Responsive, user-friendly design consistent with Singku Café’s warm aesthetic.
